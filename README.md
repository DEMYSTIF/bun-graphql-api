Clone the project

```bash
git clone https://github.com/tr1sm0s1n/bun-graphql-api.git
cd bun-graphql-api
```

Install Bun

```bash
curl -fsSL https://bun.sh/install | bash
```

Install dependencies

```bash
bun install
```

Start the application

```bash
bun run dev
```

Open [browser](http://localhost:4000) for queries and mutations.
